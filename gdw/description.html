HTML Terms

HTML
Acronym for Hypertext Markup Language, and is a powerful web development tool to create pages on the World Wide Web
HTML element
A component of a web page made with HTML, starting with a <~> tag and ending with a </~>tag
HTML tag
A modifier of text or the webpage itself. It starts with the tag itself enlcose bteween < and >, and most of the ends with a / before the tage after the <
angle bracket
A < or > sign, also called a less than and reater than sign
HTML attribute
A modifer of a tag, it goes between the tag and the > sign. It is comprised of a name and value with a equal sign and quotations between
HTML comment
Text written in HTML that only an editor of the website can view
HTML object elements
Elements in HTML that affect a paragraph itself, not the text. They also have no closing tag (includes <br>, <hr>, and <img>)
HTML special character
Characters in HTML that cannot be written as they would be normally and require a special formatting that starts with a & and ends with a ; (includes the <, >, &, and a non-breaking space, which cannot be seen in text view)
